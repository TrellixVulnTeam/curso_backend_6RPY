ESTRUCTURA DE DATOS

Una lista permite almacenar distintos tipos de datos, a diferencia de un array.

Métodos en listas
lista.append(x): agregar un elemento al final de una lista
lista.extend(lista2): juntar dos listas
lista.insert(x): inserta un elemento indicando la posición donde se desea agregar
lista.remove(x): elimina un elemento de acuerdo al valor del elemento
lista.index(i): retorna la posición de un elemento de acuerdo al valor
lista.pop(i): elimina un elemento por la posicion
len(lista): retorna la cantidad de elementos de una lista
lista.count(x): retorna el numero de veces que se repite un elemento de acuerdo al valor
lista.sort(): ordena los elementos de menor a mayor u orden alfabético o mediante condiciones
lista.reverse(): invierte el orden original de una lista
lista.clear(): remueve todos los elementos de una lista
enumerate(lista): es especialmente útil cuando al emplear un bucle for se precisa tanto de los elementos de un iterable como de su posición
range(i,j,k): rango de numeros, i posicion inicial, j posicion final menos 1, x saltos
break: detiene un bucle
